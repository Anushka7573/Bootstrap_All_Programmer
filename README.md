📌 What is Bootstrap?

Bootstrap is a free and open-source CSS framework used to create responsive and modern websites quickly.

In simple words:
👉 Bootstrap helps developers design beautiful websites without writing too much CSS.

It provides ready-made components like:

Buttons
Navbar
Cards
Forms
Modals
Grid System
Alerts
And much more
⚡ Why Bootstrap is Popular?

✅ Easy to learn
✅ Mobile responsive design
✅ Saves development time
✅ Modern UI components
✅ Works on all browsers
✅ Large community support

🛠️ Features of Bootstrap
1. Responsive Grid System

Bootstrap uses a 12-column grid system for responsive layouts.

Example:

<div class="container">
  <div class="row">
    <div class="col-md-6">Left</div>
    <div class="col-md-6">Right</div>
  </div>
</div>
2. Ready-Made Buttons
<button class="btn btn-primary">Click Me</button>

Types of buttons:

btn-primary
btn-success
btn-danger
btn-warning
3. Navbar
<nav class="navbar navbar-dark bg-dark">
  <a class="navbar-brand" href="#">My Website</a>
</nav>
4. Cards
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Bootstrap Card</h5>
    <p class="card-text">Easy UI component.</p>
  </div>
</div>
